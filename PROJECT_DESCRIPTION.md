# Recruiter-facing project description

## GitHub short description

AI-powered PDF resume analysis with n8n, structured output, a credential-free demo, testing, and an OpenAI upgrade path.

## CV bullet points

- Built an n8n workflow for PDF resume intake, text extraction, input validation, job-description matching, and automated Markdown report generation.
- Implemented both a credential-free reproducible analyzer and an OpenAI workflow with JSON-schema-validated output, error handling, tests, privacy safeguards, and technical documentation.

## LinkedIn project description

Developed a portfolio-ready AI Resume Analyzer with n8n. The workflow accepts a PDF resume and job description through an n8n form, extracts and validates the document text, performs structured job-fit analysis, and generates JSON and Markdown reports. The repository includes a no-credential reproducible demo, an OpenAI upgrade path, schema validation, error cases, unit tests, architecture documentation, and privacy guidance.

## Skills demonstrated

n8n, workflow automation, PDF processing, form triggers, binary data, JSON, OpenAI integration, prompt engineering, structured output, Python, testing, Git, technical documentation.
